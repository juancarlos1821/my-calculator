## 📌 Descripción

Calculadora web completamente funcional desarrollada con tecnologías web puras (sin frameworks). Cuenta con una interfaz limpia, moderna y responsiva que permite realizar operaciones matemáticas básicas de manera intuitiva.

---

## ✨ Funcionalidades

- ➕ Suma, resta, multiplicación y división
- 🔢 Soporte para números decimales
- 🔄 Botón de reinicio (AC) para limpiar la pantalla
- 📱 Diseño responsivo para móviles y escritorio
- ⚡ Respuesta instantánea sin recargar la página

---

## 🛠️ Tecnologías Utilizadas

| Tecnología | Uso |
|------------|-----|
| HTML5 | Estructura y semántica |
| CSS3 | Estilos y diseño responsivo |
| JavaScript (ES6) | Lógica de la calculadora |

## 🚀 Cómo ejecutar el proyecto

```bash
# Clona el repositorio
git clone https://github.com/juancarlos1821/my-calculator.git

# Navega al directorio
cd my-calculator

# Abre en tu navegador
open index.html
```

O simplemente visita la [demo en vivo](https://juancarlos1821.github.io/my-calculator/) 🌐
